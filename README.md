# ROS Autonomous Navigation System

## Project Introduction

This project is based on the ROS (Robot Operating System) framework and implements an autonomous navigation system for an intelligent mobile robot.  
The system integrates SLAM mapping, autonomous localization, global path planning, dynamic obstacle avoidance, and target point navigation.

The project was developed and optimized during the RAICOM CIAM Engineering Maker Competition and achieved excellent performance in autonomous navigation tasks.

---

## Project Features

- SLAM environment mapping
- Autonomous localization
- Global path planning
- Dynamic obstacle avoidance
- Goal point navigation
- Simulation and debugging in Gazebo and RViz
- Navigation parameter optimization based on A* algorithm

## System Architecture

The autonomous navigation system mainly includes:

1. Environment Mapping  
   Using gmapping to generate 2D environment maps.

2. Robot Localization  
   Using AMCL for real-time robot positioning.

3. Path Planning  
   Using A* algorithm and move_base for global path planning.

4. Navigation & Obstacle Avoidance  
   Realizing autonomous navigation and dynamic obstacle avoidance.

5. Simulation Verification  
   Testing and debugging in Gazebo and RViz environments.
